* [Installation](#installation)

An express app with an AI chat bot.

## Installation

This package can be used in Laravel 5.3 or higher.
You can install the package via composer:

``` bash
git clone https://gitlab.com/Fagr/chatbot/node_chatbot.git
cd node_chatbot
npm install
```

## Usage

 - Run the server:
  - MacOS or Linux `$ DEBUG=node_chatbot:* npm start`
  - Windows `> set DEBUG=node_chatbot:* & npm start`
 - Send `POST` requets to the `/chat` endpoint with payload `{"username": "your_user_name", "message": "Your message to the bot"}`
